    O que são Clients

    Voltar ao início

    Cliente no linguajar web

    Client é o dispositivo que consome a informação da internet. Um disposotivo se comunica com outro através da internet

    Ao acessar um site, a informação vai para a internet, passa pelo seridor que envia arquivos estáticos (HTML, CSS, Javascript, imagem, áudio...), volta para a internet e chega no dispositivo client

    O navegador armazena esses arquivos no dispositivo cliente (cache), e caso queira acessar o mesmo site novamente não será necessário pedir para o servidor, evitando assim um trafego desnecessário.

    Ao criar um site diferentes resoluções devem ser consideradas (responsividade ou design responsivo), também deve se considerar a compatibilidade entre navegadores ou browsers (boas práticas e testes)

    Navegadores

    Navegadores ou browsers são programas criados por empresas utilizados para abrir/ executar arquivos.

    Aplicações web

    São soluções criadas que possuem a internet como meio de comunicação entre Client X Server, não sndo necessária a sua instalação. Quando acessamos através do navegador é uma aplicação web (ex: www.youtube.com), quando é acessado por meio de um aplicativo instalado no dispositivo não é uma aplicação web

    Sobre servidores

    Os servidores servem/executam as informações solicitadas pelo client.

    Existem vários tipos de servidores:
  • Arquivos
  • Segurança (Firewall)
  • Streaming
  • E-mail

    O servidor tem um sistema operacional que tem instalado um servidor web. Esse sistema operacional faz o processamento das requisições HTTP

    Tipos de servidores

    Proxy

    É um servidor que bloqueia o acesso a certos sites. Esse servidor fornece o acesso à internet, porém com as devidas restrições. Desse modo, quando o usuário quer entrar em algum endereço in internet, o servidor proxy confere se pode conceder o acesso, se sim ele encaminha para a internet e depois para o servidor que fornece os arquivos.

    Firewall

    O firewall bloqueia o acesso de arquivos externos. É um sistema instalado no servidor

    Web server

    Site que vai para a internet, passa pelo servidor e retorna as informações

    E-mail server

    Servidor voltado para armazenamento de e-mails

    Database server

    É quando a aplicação web instalada no servidor acessa o banco de dados

    Client → Internet → Web server/Application server → Database server

    Quando um site está disponível apenas em deterninada rede é chamada de intranet

    Clients → Web server/AApplication server → Database server

    DNS (Domain Name Service)

    Servidor que detém uma lista de nomes e IP.

    Hospedagem de sites

    O que é hospedagem de sites:

    Quando colocamos os arquivos de site em um servidor que fique disponível na internet para outras pessoas acessarem. Para isso existem empresas que oferecem planos de hospedagem.

    Conexão FTP (File Transfer Protocol)

    Conexão feita para enviar e receber arquivos. Ao contratar a hospedagem o provedor fornece os dados de acesso FTP ao servidor

    Estrutura básica do HTML

    Toda tag HTML começa com "<" e termina com "/>". Também é dividida html, head e body. html contém dentro o head e o body, o head contém elementos que vão carregar antes de exibiir para o usuário e body contém elementos que aparecerão para o usuário

    Falando sobre tags

    Algumas tags

  • i - itálico
  • strong - negrito
  • input - campo que o usuário pode digitar
  • img - imagem

    • Atributos das tags

    Alguns atributos

  • id - identificador
  • style - comandos css direto no elemento como cor por exemplo
  • input type(="text", ="number", ="color" ) - o campo pode receber texto, número ou cor
  • img src="" - recebe o caminho da imagem
  • img width="" - muda o tamanho da imagem

    • Textos

    Tags para texto

  • h1 até h6 - tags para título, subtítulo...(sendo o h1 o maior e h6 o menor)
  • p - parágrafo
  • blockquote - usado para realizar citações
  • mark - marca texto
  • sup - faz ficar em cima da linha (por exemplo um número elevado ao quadrado)

    • Listas ordenadas e não ordenadas

    Listas ordenadas

    Para fazer uma lista ordenada utilizamos ol e para inserir os itens na lista utilizamos o li. Nessa lista os itens são enumerados

    Listas não ordenadas

    Para fazer uma lista não ordenada utilizamos ul e para inserir os itens na lista utilizamos o li. Nessa lista os itens são colocados em bolinhas

    Links

    Para inserir links na página utilizamos "a href =link". O link pode abrir na mesma aba ou abrir uma outra. Para abrir na mesma página utilizamos o atributo target="_self" ou não especificar nenhum atributo. Para abrir em oytra aba utilizamos target="_blank"

    Podemos também configuraar o balãozinho do clique utilizando title=""

    Para utilizar links dentro do próprio site utilizamos a href="nome_arquivo.html"

    Para utilizar links para navegar dentro da mesma aba utilizamos a href="#texto" e na parte que será direcionado utilizamos id=texto
