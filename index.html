    O que são Clients

    Cliente no linguajar web

    Client é o dispositivo que consome a informação da internet. Um disposotivo se comunica com outro através da internet

    Ao acessar um site, a informação vai para a internet, passa pelo seridor que envia arquivos estáticos (HTML, CSS, Javascript, imagem, áudio...), volta para a internet e chega no dispositivo client

    O navegador armazena esses arquivos no dispositivo cliente (cache), e caso queira acessar o mesmo site novamente não será necessário pedir para o servidor, evitando assim um trafego desnecessário.

    Ao criar um site diferentes resoluções devem ser consideradas (responsividade ou design responsivo), também deve se considerar a compatibilidade entre navegadores ou browsers (boas práticas e testes)

    Navegadores

    Navegadores ou browsers são programas criados por empresas utilizados para abrir/ executar arquivos.

    Aplicações web

    São soluções criadas que possuem a internet como meio de comunicação entre Client X Server, não sndo necessária a sua instalação. Quando acessamos através do navegador é uma aplicação web (ex: www.youtube.com), quando é acessado por meio de um aplicativo instalado no dispositivo não é uma aplicação web
